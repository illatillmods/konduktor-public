# Public Roadmap

This roadmap describes public-facing direction only. It is not a commitment to ship dates.

## Implemented Internally

- Local CLI installation and execution
- Per-repo initialization and state
- Request classification and worker routing
- Isolated Git worktrees for editing tasks
- Deterministic review gates
- Risk-aware model review policy
- Safe local merge flow
- Manual review packet generation
- Coarse worker usage-limit awareness
- VS Code task integration
- Repo inventory for prior agent/orchestration artifacts

## Near-Term Hardening

- Clearer public demo material that does not expose implementation mechanics
- More precise documentation of safety boundaries and local-state handling
- Additional live smoke coverage around worker CLI version changes
- Better surfaced diagnostics when local worker CLIs change flags or output format

## Not Planned for This Mirror

- Source publication
- Prompt publication
- Hosted service deployment
- Public datasets or worker logs
- Commercial reuse license
