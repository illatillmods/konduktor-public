# FAQ

## Is Konduktor open source?

No. This is a public documentation mirror. The implementation is private and all rights are
reserved.

## Is this a hosted AI coding product?

No. Konduktor is a local CLI workflow that runs inside the user's existing Git repositories
and invokes already-installed worker CLIs.

## Does Konduktor replace Claude Code or Codex CLI?

No. It orchestrates them. Konduktor adds routing, isolated worktrees, review records,
safety gates, repair attempts, and local merge discipline around those tools.

## Does Konduktor push code?

No. The verified implementation does not push, deploy, force-merge, or publish work.

## Does Konduktor include benchmarks?

No public benchmark is currently published. The verified public claim is test coverage of
the orchestration behavior, not model quality or performance superiority.

## Why is the source excluded?

The value is in the workflow design, prompts, routing, review, repair, and safety machinery.
Those materials are intentionally not included in this public mirror.
